# Contributing

Keep this repository focused on the skill package.

## Rules

- Do not commit real vault content.
- Do not commit `.env`, tokens, cookies, chat histories, or plugin session data.
- Keep `SKILL.md` concise and procedural.
- Put durable reference material in `references/`.
- Validate the skill before pushing.

## Validate

```bash
python3 ~/.codex/skills/.system/skill-creator/scripts/quick_validate.py .
```
