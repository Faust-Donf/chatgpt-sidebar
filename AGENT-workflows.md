# AGENT-workflows.md — 操作流程

## Ingest 流程

1. **扫描**
   - 确认资料路径。
   - 判断是否已存在 source 页。
   - 不改写 raw 原文。

2. **创建 Source**
   - 在 `wiki/sources/` 创建摘要页。
   - 提炼一句话总结、关键要点、元信息。
   - 链接回 raw 原始资料。

3. **提取 Concept**
   - 从资料中识别概念、框架、方法论。
   - 有则更新 `wiki/concepts/`，无则新建。

4. **建立关系**
   - 添加相关页面、来源、交叉引用。
   - 标记矛盾观点和适用条件。

5. **更新导航**
   - 需要主题导航时更新 `wiki/maps/`。
   - 重要入口同步到 `index.md`。

6. **记录日志**
   - 在 `log.md` 追加时间线。

## Query 流程

1. 先搜索 `wiki/`。
2. 再搜索 `raw/`。
3. 综合回答并说明来源。
4. 如果形成稳定知识，沉淀到 source/concept/map。

## Lint 流程

- P0：source 与 raw 的链接完整性。
- P1：wiki 内链、孤立页、重复页。
- P2：页面长度、更新时间、标签和结构质量。
