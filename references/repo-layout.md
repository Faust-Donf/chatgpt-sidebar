# ChatGPT Sidebar Repository Layout

```text

AGENT.md

AGENT-workflows.md
AGENT-writing.md
CHANGELOG.md
```

```text
raw/
  inbox/
  archive/
  assets/
wiki/
  sources/
  concepts/
  comparisons/
  maps/
  projects/

```

Do not hard-code topic folders like `ai-tools` unless the user explicitly wants domain-specific structure.
