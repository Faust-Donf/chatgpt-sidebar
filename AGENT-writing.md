# AGENT-writing.md — 写作规范

## 语言

- 使用中文撰写，专业术语可保留英文。
- 风格简洁、直接、信息密度高。
- 避免空泛判断和冗余修饰。

## 标签规范

使用层级标签，用 `/` 分隔。

```text
type/source
type/concept
type/comparison
type/map
type/project
type/reference

topic/{topic-name}

status/draft
status/stable
status/outdated
status/archived
```

## 写作原则

1. 一个页面只讲一件事。
2. 先定义，再展开。
3. 引用要有上下文。
4. 区分事实、观点和推测。
5. 能链接就不要复制大段原文。

## 反模式

- 把多个概念塞进一个页面。
- 复制粘贴原始资料全文。
- 只贴链接不写说明。
- 使用模糊标题，如“一些想法”“笔记”。
- 让 source 页承担长篇个人评论。
