## 工作区结构

| 层 | 路径 | 用途 |
|----|------|------|
| 原始资料 | `raw/inbox/` | 新资料入口 |
| 原始归档 | `raw/archive/` | 已处理或历史资料 |
| 附件 | `raw/assets/` | 图片、PDF、附件 |
| 摘要 | `wiki/sources/` | 原始资料的一页式摘要 |
| 概念 | `wiki/concepts/` | 可复用概念、框架、方法论 |
| 对比 | `wiki/comparisons/` | 多方案、多概念对比 |
| 地图 | `wiki/maps/` | 主题导航和知识图谱入口 |
| 项目 | `wiki/projects/` | 项目上下文和推进记录 |
| 元信息 | `wiki/meta/` | 模板、提示词、维护资料 |
